09 · Rules of the system

If you do nothing else, follow these

Always
  • Use tokens. Never hard-code hex values, pixel sizes, or font names.
  • Metadata (durations, sizes, timestamps, counts) in mono.
  • Show names in ALL-CAPS mono. Episode titles in Sentence-case sans.
  • Layer with hairline rules or coffee tiers.
Never
  • Add rounded corners. --radius: 0 is non-negotiable.
  • Use amber for hover, selection, or decoration. Amber = right now.
  • Introduce a new color outside the token file.
  • Use shadows. Or light mode. Or Inter.
